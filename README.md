# SDL3_template

SDL3 simple example for c.

Libraries

* SDL3
  * https://github.com/libsdl-org/SDL

* sdl2-compat
  * https://github.com/libsdl-org/sdl2-compat

* SDL_image
  * https://github.com/libsdl-org/SDL_image

* SDL_ttf
  * https://github.com/libsdl-org/SDL_ttf

## How to build

Makefiles provided

* linux

install libraries for the implementation you want to use

* windows

use gcc dist from msys2 (https://www.msys2.org/)

 use the MSYS2 MinGW 64-bit cmd prompt

 pacman -S to install the libs

* mac

use brew to install packages
